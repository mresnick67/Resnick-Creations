Activate

LiftKit (Activate): local-first training intelligence.

Swift-native iOS workout tracker focused on reliable logging, data portability, and practical coaching insights.

View GitHub Repo

Product Snapshot

Activate is designed around speed and trust. It prioritizes rapid set logging, clean workout history, and transparent summaries over noisy gimmicks.

  • Platform: iOS 17+
  • Current stack: SwiftUI + SwiftData
  • Planned next stack: HealthKit + OpenAI API
  • Architecture: local-first MVP (backend deferred to V2)

Design principle: logging speed over visual novelty.
One-thumb operation, facts-first summaries, and coachable signals grounded in actual workout history.

What’s Complete

Core app shell

TabView scaffold with Home / History / Exercises / Settings.

SwiftData model layer

Workout, WorkoutSet, Exercise, WorkoutTemplate plus related enums.

Exercise library

91 seeded exercises, searchable and filterable with detail views.

Workout execution flow

Start workout, add exercises, set rows, completion toggles, add/delete sets.

Rest timer capsule

Auto-start on completion with inline controls (-15s / +15s / skip).

Finish + summary

Workout persistence, duration, completed sets, and logged volume with guardrails.

History list

Completed/In Progress sections, scanable metrics, explicit status chips.

Workout detail

Read-only drill-in preserving set order with provenance and formula disclosure.
