Forecheck Fantasy

Fantasy hockey intelligence with edge.

iOS app + FastAPI backend for competitive managers who want faster adds, sharper scans, and better league decisions.

Private GitHub repository

What It Does

Forecheck combines rolling analytics, streamer scoring, and rule-based scans with a social feed so managers can discover opportunities and debate moves in one place.

  • Discover top streamers and trending players
  • Deep player detail with rolling windows (L5/L10/L20/Season)
  • Custom scan rules for targeted adds
  • Social posts, polls, comments, and notifications

Tech stack: Swift iOS client + FastAPI backend.
NHL data ingestion, Yahoo ownership sync, and server-side evaluation power the decision layer.

Architecture

Frontend

Native iOS app for discovery, drill-in detail, scans, feed, and alerts.

Backend API

FastAPI service handling auth, player data, stats windows, scans, and feed endpoints.

Data Sources

NHL Stats API + GameCenter boxscores + Yahoo ownership sync signals.

Production Base URL

https://forecheck-fantasy-api.fly.dev

Core Product Flows

  • Streamers: ranked discovery and trend monitoring
  • Player Detail: usage context, recent games, schedule, and windowed stats
  • Scans: rule composition and automated candidate evaluation
  • Community: feed + polls + comments around roster decisions
  • Alerts: unread counts and notifications for relevant updates
